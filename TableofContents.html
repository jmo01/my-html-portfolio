HTML Table of Contents

    This document teaches about the different types of headings, open and closing heading brackets.

  2. Paragraph

    This document teaches paragraph tagging for general paragraphs and sentences.

  3. Closing Tags

    This document discusses the horizontal line (effectively a line separator) and the break element (effectively a new line character).

  4. Movie Project

    This document is a small project using paragraph, heading, and horizontal line tags to create a list of favorite movies.

    This document discusses anchor tags in order to create hyperlinks within text and image elements to pull images from websites or local repositories.

  8. Birthday Project

    This document is a mini project using headers, image tags, lists, and anchor elements to create a birthday party page.

  9. File Paths

    This document discusses pulling images from different folders within the same parent folder using proper formatting and file path sourcing.

  10. Web Pages

    This document describes a website home page with hyperlinks other pages somewhere in the same parent folder; a hyperlink to an "About Me" page behind an image of me and a hyperlink to a "Contact Me" page behind its corresponding prompt.

  11. Boilerplate

    This document discusses the boilerplate code for any HTML and breaks down it's key elements. Also using "!" in any HTML file to fire up the boilerplate.

  12. Portfolio

    This document is a mini project that uses proper file path sourcing and the boilerplate to properly display the previous two mini projects.
